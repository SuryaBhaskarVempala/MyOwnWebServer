About This Web Server

This web server is built using Java and supports multiple features for handling HTTP requests. It is designed to handle a variety of static and dynamic content, providing responses to users based on different methods like GET and POST.

Key Features of This Web Server:

  • GET Requests: Handles standard GET requests to serve static HTML files such as index.html, about.html, and contact.html.
  • POST Requests: Capable of processing POST requests, including handling form submissions and extracting parameters like name or message from user inputs.
  • Thread Pool for Scalability: Uses a thread pool with a maximum of 100 concurrent threads, which allows the server to handle multiple requests from up to thousands of users efficiently.
  • Simple Routing: Handles basic routing, redirecting users to the appropriate HTML files based on URL paths like /about and /contact.
  • File Serving: It serves files with the appropriate content types, such as HTML, CSS, and text files, using the Files.probeContentType() method.
  • HTTP Response Handling: Provides detailed HTTP response headers (including status codes like 200, 404) for both file serving and form responses.
  • Efficient Resource Management: Closes resources (like sockets and streams) after handling requests, ensuring no unnecessary memory consumption.
  • Responsive Design: The front-end HTML pages are designed to be responsive and will look good on both desktop and mobile devices.

This server demonstrates how basic web servers are built and can be expanded with additional features like user authentication, advanced routing, or integration with databases. It can easily be extended to meet specific needs.
